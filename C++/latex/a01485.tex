\doxysubsection{Detailed Description}
A struct to hold all necessary information for a C\+SR. 

\begin{DoxyRemark}{Remarks}
This can be used by external clients to recreate or map to an existing C\+SR. For more information on what these arrays contain see \href{https://eigen.tuxfamily.org/dox/group__TutorialSparse.html}{\texttt{ https\+://eigen.\+tuxfamily.\+org/dox/group\+\_\+\+\_\+\+Tutorial\+Sparse.\+html}}
