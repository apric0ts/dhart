Points to perform view analysis from.

Number of rays to fire. Higher values provide more accurate analysis, but increase the runtime.
upward_fov (Optional)
Type: SystemSingle
Maximum angle in degrees above the viewer's eye level that is considered.
downward_fov (Optional)
Type: SystemSingle
Maximum angle in degrees below the viewer's eye level that is considered.
height (Optional)
Type: SystemSingle
Height of the observer. Nodes are offset this distance from the ground before analysis is performed.

Return Value

Type: ResultArray
A 2 dimensional array of results for each node and each ray fired. For example, every ray for node one is located at row 1, and the results for node 2 are located at row 2.
Remarks
Note that the number of rays fired may exactly match ray_count depending on the provided field of view restrictions. The hitpoints for each ray in the returned array can be determined using the directions from SphericallyDistributeRays(Int32, Single, Single).
