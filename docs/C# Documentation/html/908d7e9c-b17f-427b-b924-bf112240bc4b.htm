To use this library in your C# project, add a reference to the HumanFactorsCSharp.dll in the bin folder to your project visual studio.

Getting Started

Using with Unity

To use this library with Unity, drag the bin folder from the release directory into your project's Assets directory. The path from windows explorer should look like <ProjectName>\Assets\bin. If this was successful, then Human Factors should have been added as a reference in your project's references under visual studio.

For more information on using managed plugins in unity, see the "Using the DLL in Unity" section of Unity's Official Documentation.
