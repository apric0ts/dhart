Detailed Description

Standard fundamental data structures for representing space used throughout HumanFactors.

Remarks
The datatypes in the SpatialStructures namespace are used throughout the HumanFactors. For example, the GraphGenerator and VisibilityGraph both produce a Graph as output, allowing for the code to manage the Graph's internal CSR to be centralized in a single location.
