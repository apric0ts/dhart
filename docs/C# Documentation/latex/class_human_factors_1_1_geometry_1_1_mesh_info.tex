\doxysubsection{Detailed Description}
A collection of vertices and indices representing geometry. 



Stores a reference to mesh geometry in native memory. Internally, meshes are represented as a 3 by X matrix of vertices and a 3 by X matrix for indices. In the above image, the face list and vertex list are what\textquotesingle{}s held in \mbox{\hyperlink{class_human_factors_1_1_geometry_1_1_mesh_info}{Mesh\+Info}}\textquotesingle{}s Index and Vertex arrays respectively.

\begin{DoxyInvariant}{Invariant}
Will always hold a valid mesh.
\end{DoxyInvariant}
\begin{DoxyRemark}{Remarks}
Eigen is used to manage all matricies and perform quick transformations such as Rotate\+Mesh. More details on Eigen are available here\+: \href{https://eigen.tuxfamily.org/dox/group__Geometry__Module.html}{\texttt{ https\+://eigen.\+tuxfamily.\+org/dox/group\+\_\+\+\_\+\+Geometry\+\_\+\+\_\+\+Module.\+html}}
\end{DoxyRemark}

Manually delete this mesh in Unmanaged memory. 

\begin{DoxyRemark}{Remarks}
This is called automatically when the class is garbage collected, so don\textquotesingle{}t worry about needing to manually call this. It may be advantageous to deallocate large meshes at specific times for specialized purposes where the garbage collector deleting a large mesh will cause noticable lag.
\end{DoxyRemark}
\begin{DoxyReturn}{Returns}
True if the handle is released successfully; otherwise, in the event of a catastrophic failure, false. In this case, it generates a release\+Handle\+Failed M\+DA Managed Debugging Assistant.
\end{DoxyReturn}
\begin{DoxyWarning}{Warning}
This deletes the mesh in unmanaged memory! Don\textquotesingle{}t try to use this class again after calling this function!
