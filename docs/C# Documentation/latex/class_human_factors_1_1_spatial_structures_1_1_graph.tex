\doxysubsection{Detailed Description}
A graph representing connections between points in space. 

Every \mbox{\hyperlink{struct_human_factors_1_1_spatial_structures_1_1_node}{Node}} in the graph contains an X,Y,Z coordinate which can be used to represent a specific point in space. This graph internally is stored as a C\+SR for space efficency. Nodes are stored in a hashmap containing X,Y, and Z coordinates, allowing for quick indexing of s specific nodes by location alone.

\begin{DoxyInvariant}{Invariant}
1) The C\+SR maintained by this graph will always be valid. 

2) Every unique unique node in the graph will be assigned a unique id. A \mbox{\hyperlink{struct_human_factors_1_1_spatial_structures_1_1_node}{Node}} is considered unique if it has an X,Y,Z location that is not within 0.\+0001 units of any other node in the graph.
\end{DoxyInvariant}
