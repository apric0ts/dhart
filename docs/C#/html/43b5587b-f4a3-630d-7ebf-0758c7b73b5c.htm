EmbreeRaytracer Methods

The EmbreeRaytracer type exposes the following members.

Methods
  NameDescription
IntersectForDistance
Fire a single ray and get the distance to its hit and the meshID if it hit anything. If it missed, then distance will be -1
IntersectForDistances
Fire a ray for each origin/direction pair
IntersectForPoint(EmbreeBVH, Vector3D, Vector3D, Single)
Fire a single ray from origin in direction, and get the hitpoint if it intersects
IntersectForPoint(EmbreeBVH, Single, Single, Single, Single, Single, Single, Single)
Fire a single ray, and get a point in return if it intersects
IntersectForPoints
Fire a ray for each origin/direction pair. Supports 3 combinations: 1) equal # of origins / directions, 2) one direction, multiple origins, 3) Multiple origins, single direction
IntersectOccluded
Fire one or more occlusion rays in C++. Occlusion rays are faster than standard rays, however can only return whether or not they hit anything.
