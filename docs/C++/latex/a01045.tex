\doxysubsection{Detailed Description}
Forward declared to prevent clients from importing boost. 

Just deletes the \mbox{\hyperlink{a01037}{Boost\+Graph}}. This way clients don\textquotesingle{}t need any of the boost stuff (for std\+::unique\+\_\+ptr$<$\+T, Deleter$>$, T == typename/class, Deleter == user-\/supplied struct which contains an operator() for destroying the pointee)
