\doxysubsection{Detailed Description}
A graph usable with the Boost\+Graph\+Library. 

Contains a C\+SR in boost created from a graph in Human\+Factors. This is necessary for using any of the Boost\+Graph\+Library functions.

\begin{DoxyRemark}{Remarks}
None of the memebers of this class should need to be modified after the class is created. This is intended to be a pseudo-\/wrapper to make the \mbox{\hyperlink{a01220}{H\+F\+::\+Spatial\+Structures\+::\+Graph}} usable with boost. You can read more about the boost graph libary in the official boost documentation\+: \href{https://www.boost.org/doc/libs/1_73_0/libs/graph/doc/quick_tour.html}{\texttt{ https\+://www.\+boost.\+org/doc/libs/1\+\_\+73\+\_\+0/libs/graph/doc/quick\+\_\+tour.\+html}}

Boost\textquotesingle{}s graphs can be difficult to use due to the extensive use of templates. This limits their usage outside of the H\+F\+::\+Pathfinder, so there aren\textquotesingle{}t any functions to interact with the boost graph after creation.
\end{DoxyRemark}
