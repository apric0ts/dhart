\doxysubsection{Detailed Description}
A collection of vertices and indices representing geometry. 

Internally stored as a 3 by X matrix of vertices and a 3 by X matrix for indices. \mbox{\hyperlink{a00283}{Eigen}} is used to manage all matricies and perform quick transformations such as Rotate\+Mesh. More details on \mbox{\hyperlink{a00283}{Eigen}} are available here\+: \href{https://eigen.tuxfamily.org/dox/group__Geometry__Module.html}{\texttt{ https\+://eigen.\+tuxfamily.\+org/dox/group\+\_\+\+\_\+\+Geometry\+\_\+\+\_\+\+Module.\+html}}

\begin{DoxyInvariant}{Invariant}
Will always hold a valid mesh with finite members.
\end{DoxyInvariant}

First check that the size of both mesh\textquotesingle{}s vertex arrays are the same size, then compare the distance between every vertex one by one.

\begin{DoxyWarning}{Warning}
This does not compare indices at all and will return false if both meshes contain the same vertices but in a different order.
