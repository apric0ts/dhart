Create a path from start to end in C++. If no path exists, the N\+O\+\_\+\+P\+A\+TH error code will be returned, and both pointers will be left as they were before the function was called.

\hline

\hline
{\em out\+\_\+data} & Return parameter for the path\textquotesingle{}s members. \\

\end{DoxyReturn}
