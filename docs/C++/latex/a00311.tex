\doxysubsection{Detailed Description}
Custom exceptions and error codes used interally by Human\+Factors. 

\begin{DoxyRemark}{Remarks}
These were mostly created as needed for the C\+Interface. If a certain error code needs to be returned, it should have a unique exception unless one can already be found in the std library or we want to differentiate between the exceptions we thrown and the exceptions the standard library throws. 
\end{DoxyRemark}
