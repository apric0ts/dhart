\doxysubsection{Detailed Description}
Contains standard fundamental data structures for representing space used throughout Human\+Factors. 

\begin{DoxyRemark}{Remarks}
The datatypes in the \mbox{\hyperlink{a00308}{Spatial\+Structures}} Namespace are used throughout the Human\+Factors library. For example, the \mbox{\hyperlink{a00306}{H\+F\+::\+Graph\+Generator}} and \mbox{\hyperlink{a00310}{H\+F\+::\+Visibility\+Graph}} both produce a \mbox{\hyperlink{a01220}{Graph}} as output, allowing for the code to manage the \mbox{\hyperlink{a01220}{Graph}}\textquotesingle{}s internal C\+SR to be centralized in a single location. 
\end{DoxyRemark}
