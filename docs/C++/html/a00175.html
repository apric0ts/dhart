◆ HF::RayTracer::HitInfo

struct HF::RayTracer::HitInfo

Contains no information about the ray itself,

Deprecated:
Never used. Leftover from partially implemented experimental feature. This eventually became the RayResult struct seen in the C-interface. Moving the RayResult struct to the C interface was far more useful than what could be accomplished here, since it allowed the Cinterface and its clients to define their own RayRequest structs that could be filled through templates.
