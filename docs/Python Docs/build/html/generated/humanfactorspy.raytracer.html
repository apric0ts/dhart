humanfactorspy.raytracer¶

Description

Perform efficent ray intersections with geometry.¶

This library is focused on performing ray intersections with geometry. The key datatype for this is the EmbreeBVH, which is required for most functions in this package.

Raytracing involves determining where a line would intersect a mesh if it were to continue infinitely form a specific point in the space. Many of the other modules in HumanFactors make use of ray intersections in their analysis.

Classes

EmbreeBVH(geometry) An acceelerated datastructure optimized for high performane ray intersections
RayResultList(vector_ptr, data_ptr, …) A list of results from a set of ray queries.
ResultStruct A struct of results containing distance, and meshid
