humanfactorspy.spatialstructures.CalculateEnergyExpenditure¶

humanfactorspy.spatialstructures.CalculateEnergyExpenditure(g: humanfactorspy.spatialstructures.graph.Graph)[source]¶

Calculate EnergyExpenditure for every edge in a graph and store it as a new cost type.

Parameters:g (Graph) – The graph of nodes and edges to calculate this algorithm on.
Postcondition:
The graph will be updated with a new cost type containing the results of this algorithm, asccessible with the key in CostAlgorithmKeys.ENERGY_EXPENDITURE.
