\doxysubsection{Detailed Description}
A Bounding Volume Hierarchy for the \mbox{\hyperlink{class_human_factors_1_1_ray_tracing_1_1_embree_raytracer}{Embree\+Raytracer}}. 

Upon calling the constructor for this class, a Bounding Volume Hierarchy is generated for the \mbox{\hyperlink{class_human_factors_1_1_ray_tracing_1_1_embree_raytracer}{Ray\+Tracing.\+Embree\+Raytracer}}. Without this structure, the raytracer cannot be used.

\begin{DoxyRemark}{Remarks}
A more detailed description of Embree\textquotesingle{}s B\+VH is available in intel\textquotesingle{}s paper\+: Embree\+: A Kernel Framework for Efficient C\+PU Ray Tracing \href{http://cseweb.ucsd.edu/~ravir/274/15/papers/a143-wald.pdf}{\texttt{ http\+://cseweb.\+ucsd.\+edu/$\sim$ravir/274/15/papers/a143-\/wald.\+pdf}}.
\end{DoxyRemark}

\mbox{\hyperlink{class_human_factors_1_1_ray_tracing_1_1_embree_raytracer}{Embree\+Raytracer}} for functions to use this B\+VH for ray intersections. 
\end{DoxySeeAlso}
