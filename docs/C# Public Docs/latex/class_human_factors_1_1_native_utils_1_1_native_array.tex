\doxysubsection{Detailed Description}
A read-\/only array stored in unmanaged memory. 


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & The type of the array pointed to in unmanaged memory.\\
\hline
\end{DoxyTemplParams}
This type supports many of the facilities of standard arrays, however, doesn\textquotesingle{}t actually hold any memory itself. Instead it simply owns a map to the array in native memory which can be accessed through spans.

\begin{DoxyNote}{Note}
The memory stored here is immutable, since this only holds a view to an array in C++. You can use \mbox{\hyperlink{class_human_factors_1_1_native_utils_1_1_native_array_a9ea74c37970f76b067ebae91b6918c8c}{Native\+Array.\+Copy\+Array()}} to create a modifiable copy of this array in managed code.
\end{DoxyNote}
