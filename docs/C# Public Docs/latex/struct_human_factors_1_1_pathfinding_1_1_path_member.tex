\doxysubsection{Detailed Description}
A ID of a Node in a \mbox{\hyperlink{class_human_factors_1_1_pathfinding_1_1_path}{Path}} and the cost to it\textquotesingle{}s next element. 

\begin{DoxyNote}{Note}
The ID held by a pathmember corresponds to the ID of a node in the graph from the path it\textquotesingle{}s generated from. The node corresponding to this path member, stored in the graph\textquotesingle{}s nodes array at the index of it\textquotesingle{}s ID. I.\+E. if this pathmember\textquotesingle{}s ID is 1, then it points to the node at Graph.\+get\+Nodes()\mbox{[}1\mbox{]}. 
\end{DoxyNote}
