\doxysubsection{Detailed Description}
A three dimensional vector with built in utility functions. 

The \mbox{\hyperlink{struct_human_factors_1_1_vector3_d}{Vector3D}} is the fundamental type of \mbox{\hyperlink{namespace_human_factors}{Human\+Factors}} for handling position and direction values.

\begin{DoxyRemark}{Remarks}
If using another library with similar types such as unity, consider making extension methods to easily convert them to \mbox{\hyperlink{struct_human_factors_1_1_vector3_d}{Vector3D}}. 
\end{DoxyRemark}
