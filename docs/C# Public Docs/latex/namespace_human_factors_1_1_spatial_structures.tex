\doxysubsection{Detailed Description}
Standard fundamental data structures for representing space used throughout \mbox{\hyperlink{namespace_human_factors}{Human\+Factors}}. 

\begin{DoxyRemark}{Remarks}
The datatypes in the \mbox{\hyperlink{namespace_human_factors_1_1_spatial_structures}{Spatial\+Structures}} namespace are used throughout the \mbox{\hyperlink{namespace_human_factors}{Human\+Factors}}. For example, the \mbox{\hyperlink{namespace_human_factors_1_1_graph_generator}{Graph\+Generator}} and \mbox{\hyperlink{namespace_human_factors_1_1_visibility_graph}{Visibility\+Graph}} both produce a \mbox{\hyperlink{class_human_factors_1_1_spatial_structures_1_1_graph}{Graph}} as output, allowing for the code to manage the \mbox{\hyperlink{class_human_factors_1_1_spatial_structures_1_1_graph}{Graph}}\textquotesingle{}s internal C\+SR to be centralized in a single location. 
\end{DoxyRemark}
